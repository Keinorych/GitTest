Всё о GIT
Базовые команды:
git init – инициализация проекта
git add (имя файла или .) – добавить файл в очередь
git commit -m “любой текст” – фиксирует изменения  (-m означает сообщение)
git branch – используется для управления ветками репозитория
git branch -M (название) – переименовать локальную ветку
git remote add (название) (ссылка) – добавляет новое подключение к удалённому репозиторию с указанным именем и URL
git push – для отправки изменений в удалённый репозиторий
git push -u (название) (имя ветки) – для связи между локальной и удалённой веткой
